# sorting-visualizer

An interactive sorting algorithm visualizer built with React. Generates a random array of bars and animates the sorting step by step.

Live demo: https://buszkiraly86.github.io/sorting-visualizer

## Algorithms

- Bubble sort
- Insertion sort
- Merge sort
- Quick sort (Lomuto partition)
- Heap sort

## Getting started

```bash
npm install
npm start
```

Then open `http://localhost:3000`.

## Deployment

Deployed to GitHub Pages via the `gh-pages` package:

```bash
npm run deploy
```
